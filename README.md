**Lab6_FX: Умная викторина в Telegram**

Данный проект является шестой лабораторной работой, целью которой является реализация логики викторины в Telegram-боте с использованием коллекций для хранения состояния и статистики пользователей.
Основной акцент сделан на применении HashMap и HashSet для решения практических задач.

**Функциональные возможности**

	•	Игровой процесс: запуск раунда викторины по команде /quiz, проверка текстовых ответов пользователя и информирование о результате
	•	Управление состоянием: хранение текущей сессии и прогресса для каждого пользователя индивидуально
	•	Статистика: ведение счета правильных ответов и вывод личных достижений по команде /stats
	•	Уникальность вопросов: использование коллекций для исключения повторов вопросов, которые уже задавались в текущей сессии
	•	Загрузка данных: механизм импорта вопросов и ответов из внешнего файла

**Архитектура проекта**

Программа спроектирована с четким разделением логики и управления данными:

	•	QuizGame: ядро викторины, отвечающее за генерацию вопросов и проверку ответов. Данный компонент полностью изолирован от Telegram-интерфейса
	•	UserSession: класс, связывающий пользователя с текущей игрой и хранящий его персональную статистику
	•	SessionManager: компонент на базе HashMap, который управляет всеми активными сессиями пользователей
	•	TelegramBot: уровень взаимодействия с Telegram Bot API для обработки команд /start, /stop, /quiz и /stats
